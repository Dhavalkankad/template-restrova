5. How to Install

  1. Download and extract the main ZIP file.
  2. Open the extracted restrova folder.
  3. Open index.html in your browser to preview the template.
  4. Use a code editor like VS Code, Sublime Text, or Notepad++ to edit the files.
  5. Upload all files to your hosting server when ready for deployment.

6. HTML Structure

7. CSS Files

All CSS files are located inside the assets/css/ folder.

  • bootstrap.min.css – Bootstrap framework
  • all.min.css – Font Awesome icons
  • aos.css – Animate On Scroll styles
  • fancybox.css – Fancybox gallery styles
  • swiper-bundle.min.css – Swiper slider styles
  • style.css – Main theme stylesheet
  • responsive.css – Responsive media queries

Main CSS File

assets/css/style.css

8. JavaScript Files

All JavaScript files are located inside the assets/js/ folder.

  • bootstrap.bundle.min.js – Bootstrap functionality
  • aos.js – Scroll animations
  • fancybox.umd.js – Popup image gallery
  • gsap.min.js – Advanced animations
  • swiper-bundle.min.js – Slider / carousel functionality
  • main.js – Custom theme scripts

Main JS File

assets/js/main.js

11. Customization Guide

Change Logo

Replace the logo image inside the image folder and update its path in HTML if needed.

assets/images/

Change Website Title

Edit the title inside the <head> section of each HTML file.

<title>Restrova - Restaurant & Food Delivery HTML Template</title>

Change Text Content

Open the required HTML file and edit the text directly.

Change Colors

Update the main color styles inside:

assets/css/style.css

Change Images

Replace images inside:

assets/images/
